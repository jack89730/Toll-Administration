# Toll-Administration
水费管理系统（简体中文版）——Qt制作,纯C++开发

这是一款为我为学校后勤定制的水费管理系统，现在开源出来给有需要的人们，嗯...我喜欢开源^ ^

以下是此款软件的一些特性
* 采用sqlite3数据库
* 支持管理用户
* 支持管理水表(一户最多支持四块水表)
* 智能提醒未缴费项目
* 支持复合查询（按户主名、所在楼栋）*按是否缴费查询还未实现*
* 支持打印订单
